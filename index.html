Accordion

Um accordion é um componente de interface que permite expandir e recolher seções de conteúdo, facilitando a organização de informações em espaços limitados.

JavaScript é uma linguagem de programação usada principalmente para tornar páginas web interativas, manipulando elementos, respondendo a eventos e comunicando com servidores.

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas web, definindo cores, fontes, layouts, espaçamentos e animações.

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo de páginas web, como textos, imagens, links e seções.
